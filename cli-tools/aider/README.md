# Aider - Inherits Terminal Colors

Aider automatically uses your terminal's configured colors. No additional setup needed!

## Why This Is Good

✅ Aider inherits your terminal theme automatically
✅ Just install one of our terminal themes
✅ Aider will match perfectly

## Installation

1. Install a terminal theme from our collection:

```bash
# Ghostty (recommended)
cp ../../terminals/ghostty/config ~/.config/ghostty/config

# WezTerm
cp ../../terminals/wezterm/wezterm.lua ~/.config/wezterm/wezterm.lua

# Alacritty
cp ../../terminals/alacritty/alacritty.toml ~/.config/alacritty/alacritty.toml

# Kitty
cp ../../terminals/kitty/kitty.conf ~/.config/kitty/kitty.conf
```

2. Restart your terminal

3. Launch Aider - it will automatically use the Everlife colors!

## No Config Needed

Aider doesn't have a separate theme configuration. It uses the ANSI colors from your terminal, which are already configured in our terminal themes.
