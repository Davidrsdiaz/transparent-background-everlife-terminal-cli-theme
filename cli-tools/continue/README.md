# Continue CLI - Inherits Terminal Colors

Continue CLI (`cn`) automatically uses your terminal's configured colors. No additional setup needed!

## Why This Is Good

✅ Continue CLI inherits your terminal theme automatically
✅ Just install one of our terminal themes
✅ Continue will match perfectly

## Installation

1. Install a terminal theme from our collection:

```bash
# Ghostty (recommended)
cp ../../terminals/ghostty/config ~/.config/ghostty/config

# WezTerm
cp ../../terminals/wezterm/wezterm.lua ~/.config/wezterm/wezterm.lua
```

2. Restart your terminal

3. Launch Continue CLI - it will automatically use the Everlife colors!

## Configuration

Continue CLI uses YAML configuration for models and rules, but colors are inherited from the terminal.

Config location: `~/.continue/config.yaml`
