# Every Code Theme: Everlife

Every Code is a fork of OpenAI's Codex CLI with enhanced theming support.

## Features

- **Custom theme support** - Full theme customization
- **TOML configuration** - Easy theme definition
- **Multi-provider** - Works with OpenAI, Claude, Gemini, and more

## Installation

```bash
# Copy theme config
mkdir -p ~/.config/every-code
cp theme.toml ~/.config/every-code/theme.toml
```

## Theme Configuration

The `theme.toml` file defines the Everlife color palette:

```toml
[theme]
name = "everlife"
foreground = "#d3c6aa"
background = "#1e2326"
...
```

## Customization

Edit `~/.config/every-code/theme.toml` to customize colors.
