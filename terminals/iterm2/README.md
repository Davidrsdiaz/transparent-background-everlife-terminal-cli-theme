# iTerm2 Theme Setup

iTerm2 uses a `.itermcolors` profile file for color schemes. Transparency settings must be configured manually.

## Installation

### 1. Import Color Profile

1. Open iTerm2
2. Go to **Preferences** (Cmd+,)
3. Navigate to **Profiles** → **Colors**
4. Click **Color Presets...** → **Import...**
5. Select an Everlife `.itermcolors` file (create from our palette)
6. Click **Color Presets...** → Select **Everlife**

### 2. Enable Transparency

1. Go to **Preferences** (Cmd+,)
2. Navigate to **Profiles** → **Window**
3. Set **Transparency** slider to your desired level
   - Strong: 15-25%
   - Light: 25-50%
4. Check **Blur** and set radius to ~20px

### 3. Configure Font

1. Go to **Preferences** → **Profiles** → **Text**
2. Click **Change Font**
3. Select **JetBrains Mono** (or JetBrainsMono Nerd Font)
4. Set size to **14pt**
