# Wave Terminal Theme

Everlife theme for Wave terminal.

## Installation

```bash
# Create config directory
mkdir -p ~/.config/waveterm

# Copy theme file
cp termthemes.json ~/.config/waveterm/termthemes.json
```

## Features

- **Custom theme support** - JSON-based terminal themes
- **Transparency support** - Via Wave's settings
- **Full 16-color palette** - Complete Everlife colors

## Setting Theme

1. Open Wave Terminal
2. Right-click on terminal block header
3. Select **Everlife** from theme dropdown

## Transparency Settings

1. Open Wave Settings
2. Navigate to **Appearance**
3. Adjust opacity settings
