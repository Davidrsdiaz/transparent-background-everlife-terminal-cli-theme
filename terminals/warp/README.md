# Warp Terminal Theme

Everlife theme for Warp terminal.

## Installation

```bash
# Create themes directory
mkdir -p ~/.warp/themes/standard

# Copy theme file
cp everlife.yaml ~/.warp/themes/standard/everlife.yaml

# Restart Warp
```

## Alternative: Clone Theme Repository

```bash
mkdir -p ~/.warp
cd ~/.warp
git clone https://github.com/warpdotdev/themes.git
cp everlife.yaml themes/standard/everlife.yaml
```

## Features

- **Custom theme support** - Full YAML-based themes
- **Transparency support** - Via Warp's appearance settings
- **Full 16-color palette** - Complete Everlife colors

## Transparency Settings

1. Open Warp Settings (Cmd+,)
2. Navigate to **Appearance**
3. Adjust **Opacity** slider
   - Strong: 15-25%
   - Light: 25-50%
