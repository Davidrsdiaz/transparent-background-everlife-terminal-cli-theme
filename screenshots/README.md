# Screenshots

This directory contains screenshots of the Everlife theme in various terminals and CLI tools.

## Adding Screenshots

1. Take a screenshot of your terminal/CLI with the Everlife theme
2. Use the naming convention: `terminal-name-mode.png`
   - Examples: `ghostty-dark.png`, `wezterm-dark.png`, `opencode-plan.png`
3. Submit a PR adding your screenshot

## Screenshot Guidelines

- Use a consistent window size
- Show some code/output for context
- Capture both terminal content and window borders
- Prefer showing AI tool interaction for CLI screenshots
- **Include your wallpaper** - transparent themes shine with great backgrounds! The desktop aesthetic is part of the experience
- Show both the terminal and some of your desktop to demonstrate transparency
